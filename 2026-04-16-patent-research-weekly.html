AMD 计算通信并行技术

UALink 200G 1.0 规范

2025-04

创新点: AMD/Intel/Google/Meta 等联合开放加速器互连标准,每通道 200GT/s,x4 配置 800GB/s。

详细技术方案

基于 AMD Infinity Fabric 技术,支持每计算 pod 最多 1024 设备,亚 1 微秒往返延迟,功耗仅为典型以太网网络 1/3-1/2。提供标准化 API 层支持异构加速器间设备到设备消息传递。

软件实现方案

需要交换硬件和拓扑感知软件栈配合。工程师已在起草 400G 变体规范。

AMD Instinct MI300X / CDNA 4 架构

2024-2025

创新点: CPU+GPU chiplet 异构封装,Infinity Fabric 直连,专用计算/通信引擎实现并行。

详细技术方案

MI300A 将 CPU 和 GPU chiplet 封装在同一基板上,Infinity Fabric 链路实现 GPU-to-GPU 和 GPU-to-CPU 直连,无需 CPU 中转。CDNA 4 继续增强互连带宽。支持内核内缓存指令维持一致性,避免完整内核边界进行网络同步。

软件实现方案

ROCm 栈包含 RCCL (集合通信库) 和 ROC_SHMEM (PGAS 单向通信)。GPU-aware MPI 扩展通过 RCCL 路由实现最优 GPU-GPU 带宽。

CDNA 4 白皮书

GPU-Aware MPI with ROCm

2024-2025

创新点: GPU 感知 MPI 库,自动检测 GPU 缓冲区并路由到 RCCL 优化。

详细技术方案

应用直接传递 GPU 设备指针给 MPI 调用,MPI 运行时检测 GPU 缓冲区并在节点内/节点间集合操作时重定向到 RCCL。抽象通信层解耦 MPI 与底层传输 (InfiniBand, RoCE, Slingshot, Infinity Fabric)。

Intel 计算通信并行技术

CXL 4.0 规范

2025-12

创新点: 128 GT/s (PCIe 7.0),x16 通道 512GB/s,内存池超 100TB,延迟 200-500ns。

详细技术方案

引入逻辑端口捆绑(聚合),支持最多 4 个 retimer 跨多 rank 扩展,原生 x2 通道低成本选项。相比上一代能耗降低约 33%。支持 CXL.mem 和 CXL.cache 协议,实现缓存一致性内存访问。

软件实现方案

主机 OS 驱动和固件支持 CXL.memory 和 CXL.cache 协议。内存池需要 fabric manager 软件进行资源分配。

华为 计算通信并行技术

Pangu Ultra MoE: 6000 NPU 训练

2025-05

创新点: 718B 参数 MoE 在 6000 Ascend NPU 上训练,30% MFU,优化专家并行路由降低同步开销。

详细技术方案

预测性仿真建模在物理实验前选择最优模型配置。优化 MoE 路由策略减少数千 NPU 间专家并行数据交换同步开销。系统级例程增强专家分区间并行数据路由,最小化协调延迟。

软件实现方案

自定义门控网络优化减少 MoE 训练固有 all-to-all 通信瓶颈。内部内存分配器针对 NPU 集群参数和激活存储优化。

信号驱动计算通信重叠 (arXiv:2504.19519)

2025-04

创新点: Tile-wise 重叠最大化并发,信号驱动触发,波组动态调度。

详细技术方案

矩阵计算发射完成标记立即启动数据传输,消除计算暂停。执行批次动态大小平衡传输粒度和带宽效率,通过预测搜索优化。传输前数据重排创建连续缓冲区,传输后恢复。

软件实现方案

扩展 CUTLASS 矩阵例程:保留核心计算循环,将初始内存重排嵌入最终处理阶段。专用监控内核通过原子计数器跟踪进度。独立执行流分别管理计算和传输工作流。

核心洞察

1. 开放标准对抗封闭生态

UALink (AMD/Intel/Google/Meta) 和 Ultra Ethernet 代表开放联盟对抗 NVLink/InfiniBand 的新阶段,多供应商互操作性是核心驱动力。

2. MoE 通信是最大战场

华为 Pangu Ultra (6000 NPU)、CloudMatrix384 融合算子、百度张量并行 MoE,各家竞相优化 All-to-All 瓶颈,融合算子+微批调度成主流方案。

3. 计算通信重叠需硬件协同

华为信号驱动 Tile-wise 重叠、阿里双通道分离,证明仅软件优化不够,需要专用 DMA 和硬件分区实现真正并发。

4. CXL 内存池化改变游戏规则

Intel CXL 4.0 支持 100TB+ 内存池,200-500ns 延迟,将实现内存解聚,从根本上改变 AI 集群资源分配模式。
