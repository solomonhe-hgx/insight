# Patent Research Weekly Report

Weekly patent research reports on compute-communication parallelism technologies.

## Overview

This repository hosts weekly patent research reports covering:
- NVIDIA communication, MMU, and memory patents
- AMD/Intel compute-communication parallelism patents
- Chinese companies (Huawei, Alibaba, Cambricon, etc.) patents and research

## View Latest Report

The latest report is available at: https://solomon-patent-research.github.io/patent-research-weekly/

## Report Structure

- **总览与趋势** - Overview and trend analysis
- **NVIDIA** - Communication, MMU, Memory patents
- **AMD / Intel** - Compute-comm parallelism patents
- **国内公司** - Chinese company patents and research
- **技术对比** - Technology comparison matrix
