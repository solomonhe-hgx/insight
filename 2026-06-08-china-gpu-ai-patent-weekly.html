华为 (Huawei) -- 昇腾AI芯片 / 互连 / 集合通信

四芯片封装设计 (Quad-Chiplet Packaging) -- 昇腾910D核心专利

PCT/CN2024/086375 华为 2024申请 / 2025公开
核心创新: 华为申请了一种基于硅中介层 (silicon interposer) 的四芯片堆叠封装设计，通过垂直互连技术实现芯片间超高速数据传输，单封装内可集成四颗计算芯片并搭配多组HBM高带宽内存，被认为将用于下一代昇腾910D AI加速器。
技术解读:
  • 该专利采用类似台积电CoWoS-L的桥接式封装方案，使用硅中介层实现多芯片间的高密度互连
  • 垂直互连 (vertical interconnects) 实现芯片间超高速数据传输，消除传统PCB走线的带宽瓶颈
  • 单封装集成四颗计算芯片 (quad-chiplet)，搭配多组HBM高带宽内存，大幅提升AI训练算力
  • 该设计架构与NVIDIA Rubin Ultra的四芯片方案高度相似，表明华为在先进封装领域正快速追赶
  • 尽管华为在制程节点上受限，但通过先进封装技术可在系统级别弥补单芯片性能的不足
  • 该专利被视为华为突破美国芯片出口管制的关键技术路径之一
影响分析: 这是华为在先进封装领域的重要布局。四芯片封装使华为能够在不依赖最先进制程的情况下，通过chiplet集成实现与NVIDIA竞争的AI训练算力。该专利意味着华为已掌握类CoWoS的自主封装技术，对国内半导体产业链具有战略意义。昇腾910D预计2026年推出，将成为华为AI芯片路线图中的关键节点。
来源: Tom's Hardware, TrendForce, 新浪财经, Huawei Central

灵衢 (LingQu / UnifiedBus) 超节点互连协议 -- 支持8192卡无收敛互联

灵衢 1.0 / 2.0 华为 2025年3月商用 / 2025年9月发布2.0
核心创新: 华为推出灵衢 (UnifiedBus) 超节点互连协议，实现"逻辑单机器"架构，支持最大8192张昇腾卡无收敛互联。Atlas 950 SuperPoD采用零线缆 (zero-cable) 互联设计，集群总算力超50万卡至百万卡规模。协议代码已开源至openEuler社区。
技术解读:
  • 灵衢协议将大规模GPU集群抽象为"逻辑单机器"，消除传统分布式系统中复杂的网络层次
  • Atlas 900 A3 SuperPoD (384卡) 于2025年3月开始交付，已商用部署300+套
  • Atlas 950 SuperPoD支持8192张昇腾卡，单卡带宽2TB/s，集群总带宽16PB/s
  • Atlas 960 SuperPoD进一步扩展至15488张卡，面向超大规模AI训练
  • 灵衢2.0在协议层面进行了增强，从基础器件、协议算法到光电互连全面升级
  • 零线缆液冷互联设计大幅降低部署复杂度和故障率
  • 协议开源至openEuler，对标NVIDIA NVLink/NVSHMEM生态
  • 昇腾950PR芯片搭载于Atlas 350标卡，向量算力提升2倍，支持更细粒度CacheLine
影响分析: 灵衢协议是华为对标NVIDIA NVLink的战略级互连技术。8192卡无收敛互联能力使其成为全球最大规模的GPU互连架构之一。开源策略有助于构建更广泛的生态，吸引更多ISV和开发者。Atlas 950/960 SuperPoD的发布标志着华为在AI基础设施领域从芯片级到系统级的全面竞争力。
来源: 华为全联接大会2025主题演讲, 华为发布全球最强算力超节点, 超节点架构创新, C114通信网

集合通信优化方法 -- HCCL高性能集合通信库核心专利

WO2022110805A1 华为 2022公开
核心创新: 提出一种实现集合通信的方法，通过区分有通信依赖和无通信依赖的工作请求，对无依赖请求直接转发、对有依赖请求经队列管控，显著降低通信时延和资源消耗，提升分布式训练的集合通信性能。
技术解读:
  • HCCL (Huawei Collective Communication Library) 是昇腾生态中的集合通信核心组件，对标NVIDIA NCCL
  • 该专利提出的依赖感知调度机制，可自动识别通信操作间的数据依赖关系
  • 无通信依赖的工作请求绕过队列直接转发，减少不必要的排队延迟
  • 有通信依赖的请求通过队列进行精确管控，保证数据一致性
  • 该机制在大规模分布式训练场景 (AllReduce、AllGather、ReduceScatter) 中效果显著
  • HCCL已集成到CANN (Compute Architecture for Neural Networks) 软件栈中
  • 支持MindSpore、PyTorch等主流AI框架的分布式训练接口
影响分析: HCCL是华为分布式AI训练生态的通信基石。该专利的依赖感知调度机制提升了大规模集群的通信效率，是华为在计算通信并行领域的核心技术积累。随着灵衢超节点架构的推广，HCCL的性能优化对万卡级训练至关重要。
来源: Google Patents - WO2022110805A1, 昇腾社区 - HCCL用户指南, 知乎 - HCCL高性能集合通信库

数据处理装置与互连电路方法 -- 昇腾芯片互连架构专利

CN111160549A 华为 Google Patents公开
核心创新: 提供一种数据处理装置及互连电路方法，针对AI处理器中多计算单元与外部存储/通信之间的高效数据通路设计，优化chiplet间的数据调度与传输效率。
技术解读:
  • 该专利描述了昇腾AI处理器内部互连电路的架构设计
  • 优化了多个功能单元 (functional units) 之间的数据传输路径
  • 通过专用互连电路实现计算单元与内存控制器间的高带宽低延迟通信
  • 支持多芯片间的数据协同处理，为chiplet架构提供底层互连支撑
  • 该专利是华为昇腾系列芯片内部数据通路的底层基础专利之一
影响分析: 该专利覆盖昇腾芯片内部互连架构的核心设计，是华为AI芯片实现高性能计算的基础。随着chiplet架构在昇腾910D中的应用，此类互连专利的重要性将进一步凸显。
来源: Google Patents - CN111160549A

海光信息 (Hygon) -- HSL系统互联总线

HSL (Hygon System Link) 系统互联总线协议 1.0 -- 国产开放互连标准

HSL 1.0 海光信息 2025年9月发布 / 2025年Q4发布1.0规范
核心创新: 海光信息自主研发并发布HSL (Hygon System Link) 系统互联总线协议，这是一个全栈开放的互连协议，提供完整的总线协议规范、IP参考设计和指令集开放，支持万卡级互联，具备高带宽、低延迟、全局地址空间一致性和缓存一致性等特性。
技术解读:
  • HSL是海光信息推出的开放、高带宽、低延迟的互联总线协议规范，对标NVIDIA NVLink
  • 核心特性包括: 全局地址空间一致性、缓存一致性互联、自由多链路扩展
  • 协议全栈开放: 开放完整的总线协议、提供IP参考设计、开放指令集
  • 支持万卡级 (10,000+ cards) 大规模互联，满足大规模AI训练需求
  • 同时支持CXL (Compute Express Link) 一致性互连总线，兼容PCIe、SATA、以太网等标准接口
  • 2025年9月在光合组织人工智能创新大会 (HAIC) 上正式发布
  • 已发布未来三年开放路线图，由海光信息总经理沙超群主导推进
  • 截至2026年1月，海光信息共获得642项专利，其中398项为发明专利
影响分析: HSL的发布是国产GPU互连生态的里程碑事件。作为首个国产全栈开放的互连总线协议，HSL有望成为国内GPU/加速芯片互连的事实标准。其开放策略 (IP参考设计+指令集) 类似于华为灵衢的开源路线，表明国内互连生态正从封闭走向开放。海光与华为在互连标准上的竞争将推动国内互连技术的快速迭代。
来源: 经济观察报 - HSL协议详解, 雪球 - 海光HSL技术开放, 东方财富 - HSL开放分析, 海光信息官网

寒武纪 (Cambricon) -- MLU-Link互连 / Chiplet架构

MLU-Link 多芯互联技术 -- 对标NVLink的自研chiplet互连方案

MLU-Link 寒武纪 2021年首发 / 持续迭代
核心创新: 寒武纪自研的MLU-Link多芯互联技术，首次搭载于思元290智能芯片，为每颗芯片提供高带宽、低延迟的多芯互联通道。通过专用MLU-Link桥接卡实现4卡/8卡全互联，支持统一内存获取和地址映射，是国内首个量产的AI芯片chiplet互连方案。
技术解读:
  • MLU-Link于2021年随思元290芯片首次推出，对标NVIDIA NVLink互连技术
  • 思元370 (MLUarch03架构) 采用chiplet技术，单芯片封装2颗AI计算芯粒 (MLU-Die)
  • MLU370-X8加速卡集成双芯片四芯粒，通过MLU-Link实现统一内存访问
  • MLU-Link桥接卡支持4张加速卡为一组的8颗思元370芯片全互联
  • 为芯片间提供低功耗、低延时和超高带宽的互连通道
  • 支持芯片上实现统一的内存获取和地址映射，创新性地实现chiplet级别的内存一致性
  • 截至2025年6月，寒武纪累计申请专利1,831项 (含境外705项、PCT 310项)
  • 拥有完整的芯片指令集专利群，覆盖云端、边缘端、终端三条产品线
影响分析: MLU-Link是国内AI芯片领域最早量产的chiplet互连方案之一，寒武纪在互连专利上的积累为其在国产AI芯片市场提供了重要竞争壁垒。随着思元系列芯片的迭代，MLU-Link技术将持续升级，支撑更大规模的分布式训练。寒武纪在2025年计划将芯片产量提升三倍，MLU-Link的规模化部署将成为关键。
来源: 寒武纪官网 - 思元370, 寒武纪 - MLU370-X8, 智源社区 - 寒武纪思元370, 电子工程专辑 - 思元370 Chiplet

多维数据高效访问方法与装置 -- AI处理器内存管理专利

US20230289292A1 寒武纪 2023公开
核心创新: 提出一种高效访问多维数据的方法与装置，通过AI处理器中多个功能单元与片上/片下存储的协同设计，优化神经网络计算中的多维数据 (张量) 访问效率，降低内存带宽瓶颈。
技术解读:
  • 该专利针对AI推理/训练中多维数据 (tensor) 的内存访问模式进行优化
  • 通过功能单元与存储层级之间的数据通路优化，减少不必要的数据搬运
  • 支持多层级存储架构 (片上SRAM + 片下HBM/DRAM) 的高效数据调度
  • 对卷积、矩阵乘等AI核心算子的数据预取和缓存策略进行了专门优化
影响分析: 该专利覆盖AI处理器内存管理的核心技术，是寒武纪在计算-存储协同优化领域的重要知识产权。随着模型规模增大，内存带宽瓶颈日益突出，此类专利的价值将持续提升。
来源: Google Patents - US20230289292A1

壁仞科技 (Biren Technology) -- 光互连GPU超节点 / BR166

国内首个光互连光交换GPU超节点方案 -- 硅光互连技术商用落地

光互连SuperNode 壁仞科技 + 曦智科技 2025年7月 WAIC发布
核心创新: 壁仞科技联合曦智科技等合作伙伴，发布国内首个基于光互连光交换技术的GPU超节点方案。采用曦智科技全球首创的分布式光交换技术和硅光技术的光互连光交换芯片，结合壁仞科技自主原创GPU架构，实现国内首次GPU与OCS (光交换机) 通信连接，率先完成光互连技术商用落地。
技术解读:
  • 该超节点方案将光互连技术引入GPU集群，替代传统电互连，大幅提升互连带宽并降低功耗
  • 曦智科技的分布式光交换技术采用硅光 (silicon photonics) 工艺制造光互连光交换芯片
  • 壁仞科技GPU通过OCS (Optical Circuit Switch) 实现光信号直连，减少光电转换开销
  • 这是国内首次实现GPU与光交换机的通信连接，标志着光互连技术在AI算力领域的商用突破
  • 壁仞科技同时在研发3D堆叠和CPO (共封装光学) 等先进技术
  • 旗舰产品BR106和BR166已完成全形态量产与规模交付
  • 截至2025年底，壁仞科技在全球累计申请专利1,600余项，位列中国通用GPU公司第一
  • 发明专利授权率位居中国通用GPU公司前列
影响分析: 光互连被视为下一代AI算力集群的关键技术。壁仞科技率先在国内实现光互连GPU超节点的商用落地，在技术路线上具有前瞻性。相比传统电互连，光互连可提供数十倍的带宽提升和显著的功耗优势。随着AI集群规模向万卡、十万卡演进，光互连将成为必选技术。壁仞在光互连领域的先发优势为其在国产GPU竞争中建立了差异化壁垒。

摩尔线程 (Moore Threads) -- MTLink互连 / C256超节点 / 花港架构

MTLink互连技术 + MTT C256超节点架构 -- 计算交换一体化万卡训练方案

MTLink / C256 摩尔线程 2025年12月 MDC 2025发布
核心创新: 摩尔线程发布MTLink卡间互连技术 (带宽达1314 GB/s) 和MTT C256超节点架构规划。C256采用计算与交换一体化的高密设计，实现单柜256 GPU全互联。同时发布新一代"花港"架构，能效提升10倍，算力密度提升50%，支持FP4精度。
技术解读:
  • MTLink是基于MTT S4000自研的卡间互连技术，支持2卡、4卡、8卡MTLink绑定
  • MTLink互连带宽达1314 GB/s，支撑十万卡级集群的大规模训练
  • MTT C256超节点面向下一代超大规模智算中心，采用计算与交换一体化高密设计
  • 单柜集成256颗GPU实现全互联，大幅降低集群部署复杂度
  • "花港"架构是摩尔线程新一代全功能GPU架构，能效比提升10倍
  • 算力密度提升50%，新增FP4/FP6低精度支持，大幅提升推理性能
  • 摩尔线程2025年在科创板上市，成为"国产GPU四小龙"中又一上市企业
  • 全功能GPU覆盖图形渲染、AI训练/推理、科学计算、视频处理等全场景
影响分析: 摩尔线程以"全功能GPU"定位在国产GPU市场形成差异化竞争。MTLink互连和C256超节点的发布标志着摩尔线程从单卡向系统级解决方案的战略升级。花港架构在能效和算力密度上的大幅提升，显示摩尔线程在GPU架构设计上的快速进步。十万卡集群的支撑能力使其具备参与大规模AI训练竞赛的实力。

燧原科技 (Enflame) -- GCU-LARE全域互联 / 邃思芯片

GCU-LARE 全域互联技术 -- 五代迭代的AI训练集群互连方案

GCU-LARE 燧原科技 持续迭代至第五代 / 2026年1月科创板IPO获受理

  • 邃思2.0芯片采用2.5D封装 (日月光)，整合9个芯片，是国内最大的AI计算芯片之一
  • 已迭代至第五代 (GCU-LARE V5)，每一代在带宽、延迟和扩展性上均有提升
  • 燧原科技于2026年1月成为科创板IPO首单受理项目，拟募资约60亿元

关键发现

  • 1. 华为灵衢在互联规模上领先 (8192~15488卡)，已实现大规模商用部署
  • 2. 壁仞科技是唯一实现光互连商用落地的国产GPU公司，技术路线具有前瞻性
  • 3. 海光HSL的全栈开放策略 (协议+IP+ISA) 可能成为国产互连标准的有力竞争者
  • 4. 摩尔线程MTLink公布了明确的带宽数据 (1314 GB/s)，透明度较高
  • 5. 燧原GCU-LARE已迭代五代，技术成熟度在国内处于前列

共同挑战

  • 1. 各家互连标准不统一，生态碎片化，难以形成规模效应
  • 2. 与NVIDIA NVLink/NVSwitch的成熟生态相比，国产互连在软件栈和工具链上仍有差距
  • 3. 先进封装 (CoWoS级别) 产能受限，影响chiplet方案的大规模部署
  • 4. 大部分互连带宽数据未公开，难以精确评估与NVLink的性能差距
  • 5. 万卡级以上集群的通信效率优化 (计算-通信重叠) 仍需持续投入

趋势展望 -- 国产GPU/AI芯片互连技术发展方向

趋势一: Chiplet封装成为核心竞争力

华为四芯片封装专利的曝光标志着国产芯片在先进封装领域的重大突破。在美国出口管制背景下，通过chiplet集成实现算力提升成为关键路径。预计2026-2027年，国内主要AI芯片厂商将全面采用chiplet架构，先进封装能力将成为竞争焦点。

趋势二: 光互连加速商用

壁仞科技率先实现光互连GPU超节点商用，CPO (共封装光学) 技术也在研发中。随着AI集群规模向十万卡、百万卡演进，传统电互连的功耗和带宽瓶颈将日益突出。光互连有望在2026-2028年成为大规模AI集群的主流互连技术。

趋势三: 互连标准走向开放统一

华为灵衢开源至openEuler、海光HSL全栈开放，表明国产互连技术正从封闭走向开放。预计未来2-3年内，国内可能形成1-2个主流互连标准，各家在统一标准下进行差异化竞争。这将有助于解决当前生态碎片化的问题。

趋势四: 超节点架构成为标配

从华为Atlas 950/960 SuperPoD到摩尔线程MTT C256，超节点架构已成为国产AI算力的标配。计算与交换一体化、柜内全互联的设计思路，正在从"创新"变为"必需"。超节点之间的Scale-Out互连将成为下一阶段的技术竞争焦点。

趋势五: 计算-通信并行优化深化

华为HCCL的依赖感知调度、寒武纪的多维数据高效访问等专利表明，国产厂商在计算-通信并行优化上持续投入。随着MoE模型和大规模并行的普及，计算与通信的重叠 (overlap) 效率将成为衡量AI集群性能的关键指标。

趋势六: 国产GPU"四小龙"齐聚资本市场

燧原科技2026年1月科创板IPO获受理，加上摩尔线程2025年科创板上市、壁仞科技港股上市，国产GPU"四小龙"已全部进入资本市场。资本的注入将加速技术研发和产品迭代，预计2026-2027年将看到国产GPU在性能和生态上的显著进步。
