US12326820B2: 用于内存模型一致性的组播与反射式内存行为

Memory Cache Coherence MMU 授权 2025-06-10
核心创新: 使单个内存请求可同时传播到多个物理地址，支持跨设备的组播(memory multicast)操作。通过 MMU 将虚拟地址映射到多个物理地址，并利用访问权限约束维护一致性。
技术解读:
  • GPU MMU 支持组播和反射式内存操作，一次请求可同时访问多个处理元件
  • 虚拟地址可映射到多个物理地址，实现高效的数据分发
  • 支持独立或共享的组播/单播指令集，灵活适配不同场景
  • 适用于多 GPU 配置和 NVLink 互连系统中的数据广播场景
  • 对集合通信(如 AllReduce/AllGather)的硬件加速具有重大意义
影响分析: 该专利为 GPU 提供了硬件级的内存组播能力，可显著加速分布式训练中的集合通信操作。在大规模 AI 训练集群中，组播内存操作可减少网络流量和延迟，是 NVIDIA 下一代互连架构的关键技术。
来源: Google Patents - US12326820B2

US12517730B2: 数据中心/多处理器系统中的自同步远程内存操作

NVLink Synchronization Remote Memory 授权 2026-01-06
核心创新: 将 store(存储)和 sync(同步)操作合并为单一数据包，发送到"一致性点"(point of consistency)后拆分为独立的内存存储和同步操作，避免单独的 fence/ack 周期，大幅减少互连流量。
技术解读:
  • 合并 store + sync 为单一原子操作，减少处理器间通信次数
  • 在"一致性点"处拆分操作，保证内存一致性和同步语义
  • 避免了传统方案中 store → fence → ack → sync 的多轮交互
  • 直接适用于 NVLink 和多 GPU 通信模式
  • 对 DSMEM 跨 SM 通信和大规模 GPU 集群的远程内存访问至关重要
影响分析: 该专利解决了多 GPU 系统中远程内存操作的性能瓶颈。在 10 万+ GPU 集群中，减少每次远程操作的同步开销可带来显著的吞吐量提升，是 NVIDIA 数据中心级 GPU 互连的核心优化。
来源: Google Patents - US12517730B2

US12566704B2: 缓存一致性的作用域树一致性协议 (已授权)

Cache Coherence 已授权 Memory Hierarchy 授权 2026-03-03
核心创新: 通过"作用域树"(scope tree)结构动态迁移缓存层次中的一致性点(PoC)，获取子缓存的 PoC 令牌(token)并更新目录条目，将一致性点迁移到最低可见缓存层级，减少访问延迟。
技术解读:
  • 缓存持有四种状态: invalid / PoC / branch / root，形成树状层次结构
  • PoC 可动态迁移到活跃线程的最低可见缓存层级，减少访问延迟
  • 通过 PoC 令牌机制确保迁移过程的正确性
  • 相比固定一致性点的传统方案，大幅降低缓存一致性维护开销
  • 适用于多 GPU、多 SM 之间的缓存一致性管理
影响分析: 这是此前 US20250307149A1 申请的授权版本。该专利解决了 GPU 缓存一致性的核心难题，在大规模多 GPU 训练和 DSMEM 场景中，高效的缓存一致性协议是确保数据正确性和性能的关键。
来源: Google Patents - US12566704B2

US12339779B2: 慢速互连上近远端内存间的数据替换方法

Memory NUMA Interconnect 授权 2025-06-24
核心创新: 在异构 NUMA 系统中将近端内存(near memory)和远端内存(far memory)之间的工作集动态分配，使用自适应粒度(64KB~1MB)的预取器算法进行数据迁移，防止抖动(thrashing)。
技术解读:
  • 基于树状结构管理工作集，当占用率超过 50% 时触发预取
  • 自适应迁移粒度从 64KB 到 1MB，根据访问模式动态调整
  • 当占用率低于 50% 时反向执行驱逐(eviction)
  • 专为通过 NVLink 等"慢速"互连连接的异构 NUMA 系统设计
  • 解决了 CPU-GPU 混合系统中数据放置策略的优化问题
影响分析: 该专利对 NVIDIA Grace Hopper 等 CPU+GPU 超级芯片的内存管理至关重要。自适应的近远端数据迁移策略可在不增加互连带宽的情况下，显著提升内存密集型 AI 工作负载的性能。
来源: Google Patents - US12339779B2

影响分析: 该专利完善了 NVIDIA 统一内存的编程模型，使开发者可以更精确地管理和优化多 GPU 环境中的内存资源分配。
来源: Google Patents - US12474981B2

AMD 计算通信并行专利

WO2025221367A3: 片上集合操作 (On-chip Collective Operations)

Collective Comm DMA ML 公开 2026-01-29
核心创新: 在芯片上直接执行集合操作(如 AllReduce、AllGather)，无需片外通信。使用专用 DMA 电路和与主缓存层次分离的解耦缓冲区(decoupled buffer)处理集合操作数据，避免缓存污染。
技术解读:
  • 多个 DMA 电路将数组条目获取到独立于缓存的解耦缓冲区中
  • 处理电路直接从缓冲区获取操作数执行集合操作(归约、广播等)
  • 解耦缓冲区设计避免集合操作污染主缓存
  • 支持在芯片上重叠计算与数据交换，实现计算通信并行
  • 代表 AMD 在 Instinct GPU 上对标 NVIDIA NCCL 的集合通信优化方案
影响分析: 该专利是 AMD 在分布式训练通信优化领域的重大布局。片上集合操作可显著降低 MI300/MI350 系列 GPU 的通信延迟，直接挑战 NVIDIA 在集合通信软件生态(NCCL)的领先地位。
来源: Google Patents - WO2025221367A3

  • 相比平面缓存设计降低功耗

  • 适用于 AMD MI300/MI350 系列 AI/HPC 加速器的缓存层次优化
影响分析: 该专利为 AMD 下一代 GPU 的缓存架构提供了创新方案。均衡的缓存延迟对 AI 推理中 KV Cache 访问和训练中的权重加载性能有直接影响。

US20250201793A1: 用于大容量内存的垂直堆叠芯片封装架构 (Z-Angle Memory)

  • 设计为 AI 数据中心加速器中 HBM 的潜在替代方案
  • 200TB/s 带宽远超当前 HBM3E 的约 5TB/s
  • 解决了限制大模型训练和推理的内存容量瓶颈
  • 代表 Intel 对传统 HBM 堆叠范式的颠覆性替代方案
影响分析: Z-Angle Memory 如果成功商业化，将彻底改变 AI 加速器的内存架构。其 1-2TB 容量和超高带宽可能使单 GPU 节点运行更大规模的模型，减少对模型并行的依赖，从而降低通信开销。
来源: Google Patents - US20250201793A1

技术方法: 通过实证研究测量重叠对 GPU 性能的真实影响，量化通常被假设为"免费"的通信重叠的隐性成本。分析了性能和功耗两方面的影响。

技术方法: 在计算流和通信流之间使用信令机制进行协调，结合智能操作重排序，以细粒度方式最大化计算-通信重叠。

技术方法: 通信分解将集合操作拆解为细粒度的点对点传输，可与计算以更细粒度交错，实现完全的计算-通信重叠。

趋势展望

内存架构革命

三大趋势并行发展: NVIDIA 通过组播内存(US12326820B2)和自同步远程内存(US12517730B2)优化现有统一内存架构；Intel 的 Z-Angle Memory(US20250201793A1)试图颠覆 HBM 范式，实现 1-2TB 容量和 200TB/s 带宽；华为则通过无 TSV 3D 堆叠(EP4123695A1)在制裁环境下寻找替代路径。内存墙正成为各家竞争的核心战场。

来源: US12326820B2, US20250201793A1

编译器驱动的通信优化

2025-2026 年最显著的趋势是编译器级别的通信优化爆发。DeepCompile(Microsoft)、Triton-distributed(ByteDance)、TileLink(ByteDance)、Syncopate 等方案分别从不同角度实现自动化的计算-通信重叠。这标志着通信优化正从手工调优向编译器自动优化转变，大幅降低开发者的优化门槛。

来源: DeepCompile, Triton-distributed, TileLink

片上集合通信硬件化

AMD 的片上集合操作专利(WO2025221367A3)和 NVIDIA 的组播内存专利(US12326820B2)表明，集合通信正从软件实现向硬件加速演进。专用 DMA 电路、解耦缓冲区和硬件级组播将使集合通信延迟降低一个数量级，直接挑战 NCCL 等软件库的角色。

来源: WO2025221367A3, US12326820B2

MoE 通信优化成焦点

随着 MoE 架构成为大模型主流，All-to-All 通信优化成为研究热点。COMET(1.71x)、Megatron Core MoE(NVIDIA)、BigMac(3.09x)等方案从不同层面解决 MoE 通信瓶颈。MoE 的通信模式与传统数据并行有本质不同，需要专门的优化策略。

来源: COMET, Megatron Core MoE, BigMac

国内 GPU 厂商专利布局加速

壁仞科技在 GPU 内存管理(CN119536889A)和计算微架构(US12475190B2)上的授权专利、摩尔线程的跨 GPU 共享内存(EP4641381A4)和缓存优化(WO2024198789A1)，表明国内 GPU 厂商正加速在核心 IP 上的布局。壁仞已有 1600+ 全球专利申请，摩尔线程有 514 项授权专利。

来源: CN119536889A, EP4641381A4

通信减少 vs 通信隐藏

两条技术路线正在分化: 一是"通信隐藏"路线，通过更好的重叠(FlashOverlap、Syncopate)将通信延迟隐藏在计算之后；二是"通信减少"路线，通过 NoLoCo(消除 AllReduce)、部分同步激活(NeurIPS 2025)等方法从根本上减少通信量。最优方案可能是两者的结合。

来源: NoLoCo, Partial Sync TP, FlashOverlap

本周要点总结

NVIDIA: 本周发现 7 项新专利，重点在内存组播(US12326820B2)、自同步远程内存(US12517730B2)和缓存一致性协议授权(US12566704B2)。NVIDIA 正在构建从硬件级组播到编译器优化的完整通信加速栈。

AMD: 片上集合操作专利(WO2025221367A3)是最重要发现，标志着 AMD 正式进入硬件级集合通信优化领域。均衡延迟堆叠缓存和混合密度互连进一步完善了 MI350 系列的技术基础。

Intel: Z-Angle Memory 是最具颠覆性的技术，如果成功将重新定义 AI 加速器的内存架构。共封装光学和多 Tile GPU 架构则展示了 Intel 在互连和计算架构上的全面布局。

研究前沿: 编译器驱动的通信优化成为最热方向，Syncopate(4.7x)和 TileLink(20.76x)展示了巨大的优化空间。重叠开销的实证表征(平均 18.9% 减速)提醒我们需要更精细的重叠策略。
